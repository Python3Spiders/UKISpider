# 项目简介


数据抓取部分：通过 Appium 爬取当前大火的社交应用 UKI 上动态广场数据，掌握爬取 App 数据的一般流程和技巧；同时掌握典型的非关系型数据库 MongoDB 的使用；

数据分析部分：主要包括数据清洗、数据整合；并将分析结果以折线图、直方图、饼状图、词云的形式直观地呈现出来。
